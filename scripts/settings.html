You can change the look and behavior of SumatraPDF by editing SumatraPDF.dat settings file.

The file is stored in %APPDATA%\SumatraPDF directory for the installed version or in the same directory as SumataPDF.exe executable for the portable version.

The file is in a simple text format. Here's what different settings mean and what is their default value:

All values in this structure can't be modified from within the UI and are for 
experimental or expert settings. They're overwritten by the content of 
SumatraPDF-user.ini if that file exists.
UserPrefs [
  background color of the non-document windows, traditionally yellow
  MainWindowBackground = #fff200

  whether the Esc key will exit SumatraPDF same as 'q'
  EscToExit = false

  whether opening a new document should happen in an already running SumatraPDF 
  instance so that there's only one process and documents aren't opend twice
  ReuseInstance = false

  these values allow to customize the UI used for fixed page documents (PDF, 
  XPS, DjVu, etc.)
  FixedPageUI [
    color value with which black (text) will be substituted
    TextColor = #000000

    color value with which white (background) will be substituted
    BackgroundColor = #ffffff

    colors to use for the gradient from top to bottom (stops will be inserted at 
    regular intervals throughout the document); currently only up to three 
    colors are supported; the idea behind this experimental feature is that the 
    background might allow to subconsciously determine reading progress; 
    suggested values: #2828aa #28aa28 #aa2828
    GradientColors =

    horizontal and vertical distance between two pages in facing and book view 
    modes
    PageSpacing = 0 0
  ]

  these values allow to customize the UI used for ebooks (with UseFixedPageUI 
  disabled)
  EbookUI [
    whether the UI used for PDF documents will be used for ebooks as well 
    (enables printing and searching, disables automatic reflow)
    UseFixedPageUI = false

    color for text
    TextColor = #5f4b32

    color of the background (page)
    BackgroundColor = #d9f0fb
  ]

  these values allow to customize the UI used for images and comic books
  ImageOnlyUI [
    horizontal and vertical distance between two pages in facing and book view 
    modes
    PageSpacing = 0 0
  ]

  these values allow to customize the UI used for CHM documents (with 
  UseFixedPageUI disabled)
  ChmUI [
    whether the UI used for PDF documents will be used for CHM documents as well
    UseFixedPageUI = false
  ]

  this list contains a list of additional external viewers for various file 
  types (multiple entries of the same format are recognised)
  ExternalViewers [
    command line with which to call the external viewer, may contain %p for page 
    numer and %1 for the file name
    CommandLine =

    name of the external viewer to be shown in the menu (implied by CommandLine 
    if missing)
    Name =

    filter for which file types the menu item is to be shown (e.g. 
    "*.pdf;*.xps"; "*" if missing)
    Filter =
  ]

  zoom levels which zooming steps through, excluding the virtual zoom levels fit 
  page, fit content and fit width (minimal allowed value is 8.33 and maximum 
  allowed value is 6400)
  ZoomLevels = 8.33 12.5 18 25 33.33 50 66.67 75 100 125 150 200 300 400 600 800 1000 1200 1600 2000 2400 3200 4800 6400

  zoom step size in percents relative to the current zoom level (if zero or 
  negative, the values from ZoomLevels are used instead)
  ZoomIncrement = 0

  these values allow to override the default settings in the Print dialog
  PrinterDefaults [
    default value for scaling (shrink, fit, none)
    PrintScale = shrink

    default value for the compatibility option
    PrintAsImage = false
  ]

  sizes of the top, right, bottom and left margin (in that order) between window 
  and document
  WindowMargin = 2 4 2 4

  these values allow to customize how the forward search highlight appears
  ForwardSearch [
    when set to a positive value, the forward search highlight style will be 
    changed to a rectangle at the left of the page (with the indicated amount of 
    margin from the page margin)
    HighlightOffset = 0

    the width of the highlight rectangle for when HighlightOffset is set
    HighlightWidth = 15

    the color used for the forward search highlight
    HighlightColor = #ff8165

    whether the forward search highlight will remain visible until the next 
    mouse click instead of fading away instantly
    HighlightPermanent = false
  ]
]

whether to store display settings for individual documents
RememberStatePerDocument = true

the ISO code of the current UI language
UiLanguage =

whether the toolbar should be visible by default in the main window
ShowToolbar = true

whether the Favorites sidebar should be visible by default in the main window
ShowFavorites = false

if false, we won't ask the user if he wants Sumatra to handle PDF files
PdfAssociateDontAskAgain = false

if pdfAssociateDontAskAgain is true, says whether we should silently associate 
or not
PdfAssociateShouldAssociate = false

whether SumatraPDF should check once a day whether updates are available
CheckForUpdates = true

the time SumatraPDF has last checked for updates (see: CheckForUpdates)
TimeOfLastUpdateCheck = 0 0

When we show 'new version available', user has an option to check 'skip this 
version'. This remembers which version is to be skipped.
VersionToSkip =

if true, we remember which files we opened and their settings
RememberOpenedFiles = true

whether to display documents black-on-white or in system colors
UseSysColors = false

pattern used to launch the editor when doing inverse search
InverseSearchCmdLine =

whether to expose the SyncTeX enhancements via Settings/Options menu
EnableTeXEnhancements = false

how pages should be laid out by default, needs to be synchronized with 
DefaultDisplayMode after deserialization and before serialization
DefaultDisplayMode = automatic

the default zoom factor in % (negative values indicate virtual settings)
DefaultZoom = fit page

default state of new windows (same as the last closed)
WindowState = 1

default position (can be on any monitor)
WindowPos = 0 0 0 0

whether the table of contents (Bookmarks) sidebar should be shown by default 
when its available for a document
ShowToc = true

if sidebar (favorites and/or bookmarks) is visible, this is the width of the 
left sidebar panel containing them
SidebarDx = 0

if both favorites and bookmarks parts of sidebar are visible, this is the height 
of bookmarks (table of contents) part
TocDy = 0

whether to display Frequently Read documents or the About page in an empty 
window
ShowStartPage = true

week count since 2011-01-01 needed to "age" openCount values in file history
OpenCountWeek = 0

display CBX double pages from right to left
CbxR2L = false

Most values in this structure are remembered individually for every file and are 
by default also persisted so that reading can be resumed
FileStates [
  file path of the document
  FilePath =

  in order to prevent documents that haven't been opened for a while but used to 
  be opened very frequently constantly remain in top positions, the openCount 
  will be cut in half after every week, so that the Frequently Read list 
  hopefully better reflects the currently relevant documents
  OpenCount = 0

  a user can "pin" a preferred document to the Frequently Read list so that the 
  document isn't replaced by more frequently used ones
  IsPinned = false

  if a document can no longer be found but we still remember valuable state, 
  it's classified as missing so that it can be hidden instead of removed
  IsMissing = false

  whether global defaults should be used when reloading this file instead of the 
  values listed below
  UseGlobalValues = false

  how pages should be laid out for this document, needs to be synchronized with 
  DefaultDisplayMode after deserialization and before serialization
  DisplayMode = automatic

  how far this document has been scrolled
  ScrollPos = 0 0

  the scrollPos values are relative to the top-left corner of this page
  PageNo = 1

  for bookmarking ebook files: offset of the current page reparse point within 
  html
  ReparseIdx = 0

  the current zoom factor in % (negative values indicate virtual settings)
  Zoom = fit page

  how far pages have been rotated as a multiple of 90 degrees
  Rotation = 0

  default state of new SumatraPDF windows (same as the last closed)
  WindowState = 0

  default position (can be on any monitor)
  WindowPos = 0 0 0 0

  Do not modify! Hex encoded MD5 fingerprint of file content (32 chars) followed 
  by crypt key (64 chars) - only applies for PDF documents
  DecryptionKey =

  whether the table of contents (Bookmarks) sidebar is shown for this document
  ShowToc = true

  the width of the left sidebar panel containing the table of contents
  SidebarDx = 0

  tocState is an array of ids for ToC items that have been toggled by the user 
  (i.e. aren't in their default expansion state). - Note: We intentionally track 
  toggle state as opposed to expansion state so that we only have to save a diff 
  instead of all states for the whole tree (which can be quite large) - and also 
  due to backwards compatibility
  TocState =

  Values which are persisted for bookmarks/favorites
  Favorites [
    name of this favorite as shown in the menu
    Name =

    which page this favorite is about
    PageNo = 0

    optional label for this page (if logical and physical page numbers are not 
    the same)
    PageLabel =
  ]
